Flashcards
¿Qué es un section?
La etiqueta section es una etiqueta de tipo bloque que permite contener elementos de un código genérico. Se diferencia del div, en que esta etiqueta encierra una parte del código, pero no necesariamente debe tener una intención o semántica.
¿Qué es un diseño responsivo?
Un diseño web responsivo es aquel que permite escalar el mismo diseño a distintos tamaños para todos los dispositivos. Esto comenzó gracias a la necesidad de escalar diseños de páginas web a dispositivos celulares y garantizar un uso cómodo en todos los dispositivos
¿Qué es la etiqueta UL?
Las unordered lists (UL), como su nombre lo indica, son etiquetas que contienen listas no ordenadas
¿Por qué es importante tener un archivo css?
Un css permite estilizar toda el diseño de la página web, cambiandole los atributos por defecto a las etiquetas, de esta manera se permite tener una página más estilizada y acorde a la modernidad.
